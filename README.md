## Overview
This project implements an end-to-end market trend analysis pipeline for **Reliance Industries Ltd. (RELIANCE.NS)** using historical stock price data. The objective is to study price trends, volatility patterns, and short-term forecasting behavior using classical time-series techniques.

The project is designed for **educational and analytical purposes**, demonstrating how financial time-series data can be processed, analyzed, and modeled using Python.

---

## Objectives
The key goals of this project are to:
- Analyze historical price trends using technical indicators
- Study daily returns and volatility behavior
- Identify potential trend regimes (uptrend / downtrend)
- Perform short-term price forecasting using an ARIMA model
- Evaluate forecasting performance using appropriate error metrics

---

## Dataset
- **Source:** Yahoo Finance  
- **Ticker:** `RELIANCE.NS`  
- **Exchange:** National Stock Exchange (India)  
- **Frequency:** Daily  
- **Time Period:** From January 2020 onwards
